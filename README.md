# JustBB
一款不受「转评赞」限制的 MicroBlog

想说什么说什么，

爱说什么说什么。

你管我说什么呢？

中心思想是：**「论如何逃离微信」**

## 参与贡献 🤝

本项目严格遵循 [Conventional Commits](https://www.conventionalcommits.org/) 规范。
请确保您的 Commit Message 符合以下格式，以便自动生成 Changelog：

- `feat`: 新功能
- `fix`: 修复 Bug
- `docs`: 文档变更
- `style`: 代码格式（不影响功能）
- `refactor`: 代码重构
- `perf`: 性能优化
- `test`: 测试相关
- `chore`: 构建/工具/杂项

开发完成后，请运行 `npm run release` 来生成版本日志并打标签。
