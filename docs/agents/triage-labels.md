# 分诊标签

| 技能角色          | GitHub 标签       | 含义                        |
| ----------------- | ----------------- | --------------------------- |
| `needs-triage`    | `needs-triage`    | 新问题，尚待评估            |
| `needs-info`      | `needs-info`      | 等待补充复现步骤或需求细节  |
| `ready-for-agent` | `ready-for-agent` | 信息完整，可交给 agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人工决策或实施          |
| `wontfix`         | `wontfix`         | 确认不处理                  |

技能提到上述角色时，使用对应的 GitHub 标签。
