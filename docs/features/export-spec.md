# 导出与分享功能规格说明书 (Export & Share Specification)

## 1. 愿景与目标

将 JustBB 的单条 Memo 导出打造为一种**“极简桌面排版”**体验。用户不仅是分享文本，更是在创造一张具有高度审美价值的数字卡片。

---

## 2. 核心功能 (Features)

### 2.1 极致海报导出 (Poster Export)

- **动态主题系统**：提供“经典打印”、“极简禅意”、“暗夜深思”三种不同审美维度的排版。
- **物理介质模拟**：支持打孔锯齿边缘（打字机风格）与细腻噪点纹理。
- **自定义元素开关**：用户可实时控制品牌标识、日期、二维码的显隐。
- **透明圆角支持**：现代主题（圆角矩形）在导出时自动处理背景透明，完美适配各种社交背景。
- **高清分发**：支持 3x 高清渲染，并提供“复制到剪贴板”与“下载图片”双路径。

### 2.2 交互准则

- **零动效响应**：移除所有过渡动画，确保主题与开关切换的瞬间即达性。
- **内容自适应**：高度随文本量自动伸缩，确保长文本排版稳健，永不截断。

---

## 3. 技术实现要点

- **渲染引擎**: 使用 `html-to-image` 进行 DOM 到 Canvas 的转换。
- **兼容性适配**: 采用 Promise 模式的 `ClipboardItem` 解决 Safari/iOS 的异步复制限制。
- **排版引擎**: 强制 `font-variant-numeric: lining-nums` 确保数字美观。
