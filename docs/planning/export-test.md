## 2. 验证记录

### 2026-04-28 (Phase 1 交付验证)

- **环境**: Linux Desktop / Chrome
- **关键发现与修复**:
  - **修复布局抖动**: 通过固定预览容器高度和强制滚动条占位，消除了主题切换时的整体页面跳变。
  - **修复长文本滚动**: 解决了 Flex 居中导致的顶部内容无法滚动的 Bug。
  - **排版增强**: 引入 `font-feature-settings: "lnum"` 解决了数字显示不齐的问题。
  - **物理效果**: 实现了基于 DOM 元素的 36 点锯齿边缘，确保在任何截图环境下 100% 可见。
- **结论**: 视觉表现符合 @DESIGN.md 规范，美学体验显著提升。
