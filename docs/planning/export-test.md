# 导出功能测试与验证 (Export Feature TEST)

## 1. 测试用例定义

### 1.1 视觉与版式测试

| 用例 ID | 测试目标       | 预期结果                                               |
| :------ | :------------- | :----------------------------------------------------- |
| TV-04   | 比例切换适配   | 切换至 1:1 时，海报自动调整内边距并保持视觉中心对齐    |
| TV-05   | 元素隐藏逻辑   | 隐藏二维码后，底部布局应平滑收缩，不留空白死角         |
| TV-06   | 纹理叠加效果   | 开启纸张纹理后，文字边缘不应出现模糊或锯齿重叠         |
| TV-07   | 加载态布局稳定 | 点击按钮进入 Loading 时，按钮宽度/高度不应发生任何跳变 |
