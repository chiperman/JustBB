# JustMemo 开发路线图与功能清单 (Roadmap & Feature List)

> **最后更新日期**：2026-02-18  
> **状态**：进行中 (85% 完成)

---

## 1. 布局与页面结构
- [x] **三栏响应式布局**：左侧导航/热力图/标签云、中间内容流/发布框、右侧月份轴。
- [x] **移动端适配**：响应式单栏展示。

## 2. 核心功能模块

### 2.1 极简发布框 (Memo Editor)
- [x] 实时字数统计。
- [x] `Ctrl/Cmd + Enter` 快捷键。
- [x] Markdown 与图片链接解析。
- [x] 独立标签录入与联想补全。

### 2.2 内容流与渲染 (Feed & Rendering)
- [x] 瀑布流/无限滚动。
- [x] 置顶逻辑 (`is_pinned`)。
- [x] 高性能关联引用 (`@编号`) 与预览。
- [x] 分享海报生成 (含二维码)。

### 2.3 侧边栏交互 (Sidebar)
- [x] 贡献热力图 (Heatmap) 2.0 (含全量统计 Modal)。
- [x] 去年今日 (Flashback)。
- [x] 标签云 (Tag Cloud)。
- [x] 主题与字体切换。

## 3. 管理与安全

### 3.1 隐私与鉴权
- [x] 管理员登录 (Supabase Auth)。
- [x] 隐私状态切换 (`Public` / `Private`)。
- [x] 基于口令的私密内容解锁。
- [x] 数据库 RLS (Row Level Security)。

### 3.2 管理操作
- [x] 原位编辑与软删除 (垃圾箱)。
- [x] 数据一键导出 (JSON/Markdown)。
- [ ] AI 翻译 (待实现)。
