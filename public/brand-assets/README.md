# JustMemo 品牌资源

本目录保存基于 A 方案「云容器」生成的品牌资产。

- `logo.svg`：主 Logo SVG 真源。
- `favicon.svg`：小尺寸优化版 SVG。
- `icon-*.png`：从 SVG 自动导出的 PNG 图标。
- `favicon.ico`：包含 16px 和 32px PNG 的 ICO。
- `apple-touch-icon.png`：180px Apple Touch Icon。
- `og-image.png`：1200x630 分享预览图。

当前资源已接入 `manifest.json`、App Router metadata 和侧边栏品牌标识。
